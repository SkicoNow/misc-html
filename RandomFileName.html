Random Media File Name Generator

I mostly wrote this program to search for these random file names in sites like YouTube and Google and see if it would produce any interesting search results. I originally attempted to write this program in Python in 2019 but I never completed it, so in 2021 (?), I decided to finish it in HTML/JavaScript. I then recently decided to rewrite my JavaScript code to put my current JavaScript knowledge to the test and get in the habit of writing scripts again.

Selecting one of the radio buttons will change the URLs of each item in the list to take you to the selected site. Click the Generate button to produce a new set of random file names.
